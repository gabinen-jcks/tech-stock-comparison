# Tech Stock Comparison

## Project Overview

This repository contains a comparative financial analysis of two major technology companies: **Apple Inc. (AAPL)** and **Microsoft Corporation (MSFT)**. The analysis includes:

- Company overview information (sector, industry, market cap)
- Price history summary for the period 2024‑01‑01 to 2024‑06‑30
- Key financial highlights from the annual income statements
- Major shareholders breakdown
- Analyst sentiment (upgrades/downgrades) over the past 12 months
- Recent news headlines

The data collected here can be used for academic projects, investment research, or general insight into the performance of leading tech stocks.
